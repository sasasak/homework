# 마크다운 Markdown에 대해서

마크다운이란 텍스트 기반의 마크업언어로 2004년 존 그루버에 의해서 만들어졌다. 쉽게 읽고 쓸 수 있으며 HTML로 변환이 가능하다.  
특정 소프트웨어에 종속되지 않고 에디터나 플랫폼을 바꿔도 파일을 그대로 사용할 수 있다. 또한 다양한 형식으로 쉽게 변환할 수 있어 활용도가 높다.  
간단하게 쓰고 어디서든 읽히며 쉽게 변환되는 문서 작성이 가능하기 때문에 널리 사용되고 있다.

1. 장점

- 마크다운은 문법이 쉽고, 관리가 간편하다.
- 지원 가능 플랫폼 및 프로그램이 다양하다.
- 렌더링 되지 않은 상태에서도 내용을 파악하기가 쉽다.

2. 단점

- 기본 문법 외에도 다양한 변형이 존재해 플랫폼마다 다르게 랜더링되거나 오류가 발생할 수 있다.
- 모든 HTML의 마크업을 대신하지는 못한다.

# 마크다운 문법

- **제목을 만들때**에는 단어나 문장구 앞에 #을 추가해서 폰트의 크기를 설정할 수 있습니다.

> #기호가 많을수록 작아지고 총 6개까지 사용할 수 있습니다.

# 또는 제목 아랫줄에 \==를 적어놓으면 현재 문장처럼 #을 사용했을때와 똑같이 제목의 사이즈가 커지게 되고,

## \- 기호를 두 번 사용하게 되면 ##과 동일한 사이즈로 제목이 커지게 됩니다.

---

- \--- 이나 \*\*\*, \_\_\_ 를 사용하게 되면 아래에 수평선이 생기게 됩니다.

---

- #마크다운언어는 #기호 뒤에 공백을 두지 않는다면 인식이 되지 않으므로 주의해야합니다.

- **문단**을 만들려면 공백을 이용해 텍스트를 구분합니다.

- 줄 바꿈이나 새 줄을 만들려면 줄을 두 번 이상의 띄어쓰기로 끝낸 후 Enter를 입력합니다.

> ###### 그러나 편집기에서는 띄어쓰기를 확인하기 어렵고 많은 사람들이 실수로 공백을 두 번 넣는 경우가 많았기 떄문에 HTML을 지원하는 마크다운 애플리케이션의 경우 HTML 태그인 `<br>`을 사용할 수 있습니다 <br>

- 텍스트 강조를 위해서 텍스트를 **굵게** 또는 _기울임체_ 로 만들 수 있습니다.

- **굵은 강조 글씨 표현 방법**은 단어나 문장구의 앞 뒤로 별 두개나 언더바 두개를 사용해주면 됩니다. \*\*강조할 글씨\*\*

> ###### **중간 단어를 강조하고 싶은 경우에는 언더바가 아닌 별표를 사용합니다. 언더바의 경우 호환성의 문제가 있을 수 있습니다.**
>
> > ###### 예시) 이것은 \*강조할\* 단어입니다

- 텍스트를 _기울임체_ 로 표시하기 위해서는 굵은 강조 표시와 비슷하게 별이나 언더바를 사용하지만 **기호를 한 개**만 사용해주면 됩니다. 중간 단어를 변경하고 싶은 경우도 규칙은 동일합니다.

> ###### 예시) _기울임체 표현방법_
>
> ###### 예시) 이것은 _기울임체_ 입니다.

- **기울어져 있으며 굵은 강조 글씨 표현방법**은 별이나 언더바를 세 개 사용하는 것입니다.

  > ###### 이 경우에도 중간 단어를 강조하려면 별표만 사용하면 됩니다.

- **인용문**
  > 인용문을 만들기 위해서는 문장의 앞쪽에 >를 추가합니다. 여러 문단이 있는 인용문의 경우에는 문단 사이의 공백문에 똑같이 >를 추가해주면 됩니다.

> ###### 예시) 안녕하세요 이것은 인용문입니다

- **중첩된 인용문**의 경우 중첩하려는 문단 앞에 >>를 추가해주면 됩니다.

> ###### 예시) 이것은 인용문입니다
>
> > ###### 예시) 이것은 중첩 인용문입니다.
> >
> > > ###### 예시) 계속해서 인용이 가능합니다.

- **순서가 있는 목록**을 만드는 방법은 숫자 뒤에 마침표가 붙은 항목을 추가해주면 됩니다.  
  1\. 이렇게 작성하면 순서가 있는 목록이 생성됩니다.

> ###### 예시)
>
> ###### 1. 멋쟁이
>
> ###### 2. 사자처럼

> > 여기서 **_주의해야할 점_** 은 목록을 만들고 싶지 않았는데 어쩌다 보니 숫자 뒤에 마침표가 붙어버리는 상황에서는 목록이 만들어지게됩니다. 그럴때는 마침표 뒤에 **백슬래시(\\)** 를 사용해서 방지할 수 있습니다.
> >
> > ###### 예시) 1\\. 백슬래시+마침표 = 마침표를 문자취급

- **순서가 없는 목록**을 만드는 방법은 목록이 될 문장 앞에 -, \*, + 중 하나의 기호를 붙이는 것입니다.

- 순서가 있는 목록과 없는 목록은 복합적으로 사용이 가능합니다.

예시)

1. 안녕하세요
   - 저는 김근영입니다.
   - 잘 부탁드립니다.

- **이미지**를 집어넣는 방법은 문장의 맨 앞에 \!\[이미지를 설명할 대체텍스트]\(이미지경로) 방식으로 사용해주면 됩니다.
  > ###### 예시) ![고양이입니다.]("src/assets/profiles/cat.jpeg")

* 단어나 문장구를 **코드로 표시**하려면 백틱으로 묶으면 됩니다 \`코드로 표시할 문장\`

  > ###### 예시) `hello world`

* 코드로 표시하려는 단어나 구문에 백틱이 포함되어 있으면 이중 백틱으로 묶어주면 됩니다.  
  코드로 \`표시할\` 단어
  > ###### 이것은 `code` 입니다.

- **URL을 하이퍼링크 텍스트로 표시**하려면 텍스트를 대괄호로 묶고 그 바로 뒤에 괄호로 URL를 입력합니다.

  > ###### 예시) \[네이버\](http://naver.com)
  >
  > ###### 결과) [네이버](http://naver.com)

- **URL이나 이메일주소 자체를 하이퍼링크**로 표시하려면 각괄호로 묶어주면 됩니다 <이메일>

  > ###### 예시) <Ilove@likelion.com>

- 이미지에 링크를 추가하려면 \[!\[대체텍스트]\(사진경로)]\(URL) 형태로 사용해주면 됩니다.

  > ###### [![네이버]("src/assets/profiles/Naver.jpeg")](http:/naver.com)

- 마크다운 문서에서 사용되는 다양한 기호들을 기능 없이 그대로 문자로 사용하고 싶다면 이스케이프 문자인 백슬래시(\\)를 기호 앞에 사용해주면 됩니다.

> ###### 예시) \* 백슬래시가 없었다면 이 문장은 순서 없는 목록이 됩니다.

> > ###### - 백슬래시가 없다면 이렇게 됩니다.

- **취소선**  
  단어 앞 뒤로 물결 두 개를 사용해주면 취소선을 그을 수 있습니다. \~~취소할 문장\~\~

  > ###### 예시) ~~취소하겠습니다~~

- **체크리스트**  
  작업 목록 앞에 \- \[ ] 작업 목록 (대괄호 사이에는 공백이 한 칸 있어야합니다.) 형태로 적어주면 체크리스트를 만들 수 있습니다.

###### 예시)

- [x] 과제하기
- [x] 스터디하기
- [ ] 놀기
